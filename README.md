## Usage

1. Clone the repo.

``` bash
git clone https://github.com/Tijani127/PhysEngine.git
```
2. Enter the directory

``` bash
cd PhysEngine
```

3. Run it!

``` bash
node server.js
```

You now have a server running on http://localhost:3002 !


## Syntax

There is a docs panel to teach you the syntax, but i will just teach you the basic commands. 

First, DO NOT TRY TO CREATE VARIABLES. You will get errors in the console saying "Error: Expected LPAREN but got NUMBER". I might fix this in the future.

1. rect(x, y, w, h)
2. cicle(x, y, r)
3. print(num or string) // num can be 1234567890, string can be abcdefghijklmnopqrstuvwxyz
4. gravity(x, y) // default is 0, 500
